Web Publishing Documentation Guide

HTML Overview

HTML stands for HyperText Markup Language, a markup language that defines the structure and meaning of web documents.

HTML does not handle logic; its core purpose is to accurately represent the meaning of content.

Role of HTML

HTML clearly distinguishes the structure of content, such as headings, paragraphs, lists, links, and images.

Semantic Markup

Using semantic tags significantly improves accessibility and SEO quality.

Representative Semantic Tags

Tags such as header, main, section, article, and footer constitute the semantic structure.

CSS Overview

CSS plays the role of applying visual styles to the structure written in HTML.

Basic Composition of CSS

It consists of selectors, properties, and values, and priority is determined by the declaration method.

Layout System

Flexbox and Grid are core tools for modern web layouts.

Flexbox

It is suitable for 1-dimensional layouts and has strengths in alignment and spacing control.

Grid

It is optimized for 2-dimensional layouts and is advantageous for complex UI composition.

JavaScript Overview

JavaScript grants dynamic behavior and logic to web pages.

DOM Manipulation

Through the DOM API, you can select HTML elements and control events and states.

Module System

Using ES Modules allows for a clear separation of code structure.

Advantages of Modules

Maintainability, reusability, and readability are greatly improved.

Document UI Design

Document-type UI is a service type where information delivery and navigation are most important.

Role of TOC

The TOC helps quickly grasp the document structure and navigate to desired locations.

Search UX

The search function significantly reduces user fatigue in long documents.

Keyboard-based Navigation

Search and navigation utilizing shortcut keys provide an advanced user experience.
